# Karental - Rent a car

## Tech stack
* Build tool: maven >= 3.9.0
* Java: 17
* Framework: Spring boot 3.4.2
* DBMS: MySql

## Prerequisites to run application
* Java JDK 17
* docker engine

## run necessary Docker containers
  `docker compose up -d`


## Start application
`mvn spring-boot:run`

## Build application
`mvn clean package`
